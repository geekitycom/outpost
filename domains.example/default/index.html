Your server is running. This is the default domain — the fallback served whenever no other domain folder matches the request.

To publish your own content:

  1. Add a folder named after your domain, e.g. domains/example.com/.
  2. Drop an index.html (or Markdown / OPML) file inside it.
  3. Point that domain's DNS and reverse proxy at Outpost.

Edit or replace this page at domains/default/index.html.
